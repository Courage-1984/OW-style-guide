06 — Brand

Vision, Mission & Voice

Aligned to the OW Brand Guidelines (2022): vision and mission language below is taken from the official brand guidelines, while UI examples in this dashboard are implementation-ready interpretations.

Vision

It is the vision of Open Window to be a leader and trendsetter in the education sector of South Africa and internationally by achieving product excellence through academic rigour and industry relevance, the use of cutting-edge technology and the continuous upgrading of infrastructure in a customer centric environment.

Mission

Our mission is to deliver creative and critical thinkers who possess the technical, dynamic, professional and multi-skilled capabilities globally required by the creative industries.

We provide relevant education opportunities through the application of:

  • Superior educational models
  • Cutting-edge technology
  • International competitive outcomes
  • Standards of excellence
  • Industry requirements

Logo Usage Rules

Use the official landscape logo consistently. Maintain clear space around the logo and avoid stretching, skewing, or placing it on low-contrast backgrounds. The grids below are visual guides for alignment and spacing.

Correct — Dark Background
Aligned + Clear space
Open Window
Use on brand dark surfaces (#232323 / #1A1A1A). Keep consistent sizing and padding.
Correct — Light Background
Clear space guide
Open Window
Use an approved dark/black logo variant on light backgrounds (avoid low-contrast grey).
Incorrect — Distorted / Low Contrast
Don’t
Open Window
No stretching, skewing, recolouring, or placing over busy/low-contrast areas.
✓ Do
  • →Use the primary landscape logo for website headers and wide digital layouts
  • →Use the secondary icon/monogram for small applications (icons, stamps, thumbnails)
  • →Apply logo variations for both dark (#232323) and light (#EDEDED) backgrounds
  • →Use black/white logo variants where full color is not suitable
  • →Keep usage consistent across website header, letterheads and banners
✗ Don't
  • →Use the secondary icon as the default header logo on landscape layouts
  • →Apply only one-color treatment when the full brand palette is required and legible
  • →Place low-contrast logo versions on mismatched backgrounds
  • →Deviate from approved palette values (#F1B33A, #27A9DD, #AA6CAD, #E75A33)
  • →Use unapproved typefaces when Avenir Next or approved web fallback is available

09 — Layout

Lines, Margins & Spacing

These layout rules keep pages consistent across the WordPress build. Use the line treatments, content widths, and spacing scales below whenever creating a new page or template.

Line Elements
Page Title Example

The primary brand line sits directly under the first H1 on the page. It uses the brand orange, matches roughly 80% of the title text width and helps anchor the start of the content.

Line Settings
Color: var(--ow-color-tech) / #E75A33
Height: 3px
Width: ≈80% of H1 text width
Alignment: left, aligned to H1 text
Spacing above: 8px below H1
Spacing below: 24px before body copy
Header & Footer Gradient Lines (Proposal)

Thin gradient lines can be added to the top and bottom of key templates (e.g. home, hub pages) to subtly frame the content and carry the four-school colour story across the layout.

Implementation suggestion for WordPress: add these as full-width container backgrounds in the header and footer templates (via theme files or Elementor templates) so they appear consistently across the site.

Margins & Spacing
Content Width ≈ 80% viewport
  • Page content should sit in a centred column of roughly 80% of the viewport width.
  • Keep a minimum of 24–32px vertical space between stacked components.
  • Use larger spacing (40–64px) between major sections like hero → body → footer.
WordPress / Elementor Guidelines
Page container width: 80% viewport (max ~1200px)
Section padding (desktop): 80px top, 80px bottom
Section padding (tablet): 56px top, 56px bottom
Section padding (mobile): 40px top, 40px bottom
Column gap (content blocks): 24–32px
Row gap (cards / grids): 16–24px
Hero / Cover Image
Page Title or Campaign Message
Short supporting line, max 1–2 sentences.

The hero / cover image spans the full width of the viewport but is shallow in height so content is visible above the fold.

Width: 100% viewport (edge-to-edge)
Height (desktop): ≈300px
Height (tablet): ≈260px
Height (mobile): ≈220px
Content max-width inside hero: 80% viewport, centered
